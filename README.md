# number-formatter
=========

A small library that adds commas to numbers

## Installation

	`npm install @cyio/number-formatter`

## Usage

	var numFormatter = require('@cyio/number-formatter')
	var formattedNum = numFormatter(233)

	Output should be `35,666`
